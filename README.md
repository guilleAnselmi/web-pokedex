# web-pokedex
A web pokedex where can find all pokemons and see details in the pokedex

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### For test build in local 

```sh
npm run preview
```



### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
